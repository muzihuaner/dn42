发送 Peer Request

请按照以下格式准备您的 Peering 申请信息：

  • 1
    Node 想要对等互联的节点。
  • 2
    Public Key 你的 WireGuard 公钥。
  • 3
    Endpoint 你的 WG 公网信息（若为动态/无公网请说明）。
  • 4
    ASN 你的自治系统编号。
  • 5
    Sessions IPv6 + IPv4 / 仅 IPv6 / 仅 IPv4。
  • 6
    Capabilities MP-BGP / Extended next hop / Extended messages。
  • 7
    IP Addresses 对应会话的 IP（MP-BGP 且无 ENH 时需双栈）。
